REQUIREMENTS
------------
* Docker & Docker Compose
* Make

INSTALLATION
------------

### Install via Make

First you need to run docker environment
```
make run
```
After you should to build the project
```
make build
```

Now project accessible at ```http://localhost:8000```

Also you can run tests
```
make tests
```

Enjoy!
